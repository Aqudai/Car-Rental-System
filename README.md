=== Car Rental System ===

Функционал:

1. Управление автомобилями (Manage Cars):
    - Добавление новых автомобилей
    - Редактирование существующих
    - Удаление автомобилей
    - Аренда автомобилей клиентам

2. Управление клиентами (Manage Customers):
    - Добавление новых клиентов
    - Редактирование информации о клиентах
    - Удаление клиентов

3. Управление арендами (Manage Rentals):
    - Просмотр всех аренд
    - Возврат автомобилей
    - Продление аренд
    - Удаление аренд
    - Фильтрация по статусу

Как использовать:

1. Сначала добавьте автомобили и клиентов
2. Для создания аренды:
    - Откройте "Manage Cars"
    - Выберите автомобиль
    - Нажмите "Rent Car"
    - Выберите клиента и даты
3. Для управления арендами:
    - Откройте "Manage Rentals"
    - Выберите аренду
    - Используйте кнопки для действий

Данные автоматически сохраняются в базе данных SQLite (файл db/cars.db)
